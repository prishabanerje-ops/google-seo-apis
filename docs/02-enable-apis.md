# Step 2: Enable the APIs

**Goal:** Turn on PageSpeed Insights and Google Search Console APIs.

**Why:** Google APIs are disabled by default. Each one must be enabled before
your credentials can call it.

---

## 2.1 Open the API Library

In [Google Cloud Console](https://console.cloud.google.com):

1. Click the hamburger menu (☰) → **APIs & Services** → **Library**

> **Screenshot placeholder:** APIs & Services > Library menu path

## 2.2 Enable PageSpeed Insights API

1. In the search box, type `PageSpeed Insights API`
2. Click **PageSpeed Insights API** in the results
3. Click **Enable**

> **Screenshot placeholder:** PageSpeed Insights API page with Enable button

## 2.3 Enable Google Search Console API

1. Go back to the Library (click **← Library** in the breadcrumb)
2. Search for `Google Search Console API`
3. Click **Google Search Console API**
4. Click **Enable**

> **Screenshot placeholder:** Google Search Console API page with Enable button

---

## Verify

Go to **APIs & Services** → **Enabled APIs and services**.
Both APIs should appear in the list.
